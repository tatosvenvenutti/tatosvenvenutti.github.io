Unreal Editor for Fortnite

Unreal Editor for Fortnite (UEFN) is a PC application available through the Epic Games Launcher that enables developers to create and publish custom experiences within Fortnite. Fortnite and UEFN work together to allow developers to integrate new features directly into the live game. UEFN, combined with the Verse programming language, enables creators to build experiences across PC and consoles.
