Star Wars Narrative and Roleplaying

My latest work was The Narrative and Roleplay template which demostrates players how to use character devices, NPC Attitude stats, and using Verse code to create custom devices. My role was to create a custom dialog device that allows the player to interact with characters. Depending on what you say it could initate the quest, progress the quest, or change your reputation with the characters. Addition to that I also created a quest manager that is used to connect the all the dialog device, update the current objective, give out reputation to the player, and ect.

Unreal Editor for Fortnite

Unreal Editor for Fortnite (UEFN) is a PC application on the Epic Launcher that allows developers to publish custom experiances and games into Fortnite. Fortnite and UEFN work together allows user to use new features directly from the game. UEFN and the Verse language allows creators to make experiances across PC and consoles.
