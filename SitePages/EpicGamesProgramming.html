Unreal Editor for Fortnite

Unreal Editor for Fortnite (UEFN) is a PC application on the Epic Launcher that allows developers to publish custom experiances and games into Fortnite. Fortnite and UEFN work together allows user to use new features directly from the game. UEFN and the Verse language allows creators to make experiances across PC and consoles.
