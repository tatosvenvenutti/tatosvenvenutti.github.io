Bird Wars is a top down shoop 'em up where you play as a peguin attacking enemy pigeons, inspired by Space Invaders. As a gameplay programmer for this project my focus was implementing core gameplay systems, including enemy behavior, player movement, firing mechanics, input handling, and collision systems.

Enemy Logic

The EnemyLogic function processes enemy entities, handling both movement and collision. Enemies are removed upon collision; otherwise, their positions are updated according to their movement behavior.

  • Developed collision systems to detect enemies that collide with objects
  • Updating enemy behavior for movements to replicate space inavders
  • Processed enemy entity using ECS (Entity Component System) architecture

Player Logic

The Player Logic function handles responsive player input and core gameplay interactions. This function also governs shooting mechanics, regulating the firing rate.

  • Programmed player interactions, including movement and shooting mechanics
  • Handled collision detection systems for the player
